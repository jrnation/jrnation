THIS IS A GUIDE TO WINDOW MANAGERS, I AM NOT FORCING YOU TO USE ONE

Window Managers: Ditch the Mouse

Switching to a WM is like Rock Lee dropping his training weights. | 👤 By JR Nation

Right now, you are probably using a Desktop Environment (DE) like GNOME, KDE Plasma, or Cinnamon. A DE is a complete package. It comes with a start menu, a taskbar, a Bluetooth manager, a settings app, and a file browser. It is comfortable, heavy, and assumes you want to use your computer like a normal person.

But you are not a normal person anymore. You are a Power User. You know the terminal. You know your scripts. It's time to strip away the bloat and install a Window Manager (WM).

1. DE vs. WM: The Great Schism

A Window Manager does exactly one thing: it draws windows on your screen. That is it.

If you install a pure WM and log in, there is no start menu. There is no WiFi icon. There is no wallpaper. You just get a black screen. If you want a wallpaper, you have to write a script to draw one. If you want a taskbar, you have to install a separate program (like Waybar or Polybar) and configure it in plain text.

Why would anyone do this? Performance and Speed.

  • GNOME uses about 1.5 GB of RAM just sitting idle on your desktop.
  • A custom Window Manager can use as little as 150 MB of RAM.

2. Stacking vs. Tiling

Windows and macOS use Stacking window management. You drag a window with your mouse, it floats on the screen, and you stack other windows on top of it. This creates a messy desk where you spend half your time resizing boxes trying to find what is underneath.

Power users use Tiling. In a Tiling Window Manager (TWM), windows never overlap. When you open a terminal, it takes up 100% of the screen. When you open a web browser, the screen splits perfectly 50/50. Open a third app, and it mathematically splits the remaining space. Zero wasted pixels. Zero dragging and dropping.

3. The Big Three: i3, Sway, Hyprland

If you want to ascend to the TWM lifestyle, these are the modern titans:

  • i3wm: The absolute classic. It runs on the older X11 display server. It is rock-solid, uses almost zero resources, and is the best place for a beginner to start learning how to tile.
  • Sway: A drop-in replacement for i3, but built for the modern, highly secure Wayland display server. If you have an AMD or Intel GPU, this is the gold standard for stability.
  • Hyprland: The current king of the r/unixporn subreddit. It brings fluid, buttery-smooth animations, blurred transparent windows, and gorgeous eye-candy to the tiling world. It requires a bit more hardware power but looks absolutely incredible.

4. The "Mod" Key (Keyboard Driven)

In a TWM, you almost never use your mouse. Everything is controlled by a designated "Modifier" key (usually the Windows/Super key or the Alt key).

Example TWM Shortcuts:

Mod + Enter — Spawns a new Terminal instantly.

Mod + Shift + Q — Kills the currently focused window.

Mod + Arrow Keys — Shifts your focus between the split tiles.

Mod + 2 — Instantly throws you to Workspace #2.

Mod + M — Quits from Hyprland and logs you out.

⚠️ THE TRAP: Before you log into a new Window Manager, you MUST know the default keyboard shortcut to open a terminal (usually Mod+Enter). Because there are no menus, if you log in and don't know the shortcut to open a terminal to edit your config file, you will be permanently trapped staring at a black screen until you force reboot your PC.

Mastering a tiling window manager turns your keyboard into an extension of your brain. You will never want to use a floating, clicking desktop environment ever again.
